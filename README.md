# 🚗 Indian Passenger Vehicle Market Analytics Platform

An end-to-end analytics platform for the **Indian Passenger Vehicle Market**, built using **Streamlit, PostgreSQL, and Plotly**. The platform analyzes **2016–2026 passenger vehicle sales data** across **20 leading automobile brands**, providing interactive dashboards, market insights, and sales forecasting.

🔗 **Live Demo:** https://your-streamlit-app.streamlit.app

---

## 📌 Overview

This project helps users explore and analyze the Indian passenger vehicle market using interactive visualizations and data-driven insights. It covers both **Internal Combustion Engine (ICE)** vehicles and **Electric Vehicles (EVs)**, enabling comparisons across brands, models, categories, and market trends.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/bandhav100/Indian-Automobile-Market-Analytics.git
```

Move into the project

```bash
cd Indian-Automobile-Market-Analytics
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run dashboard/app.py
```
